SportSmart

A comprehensive sports equipment e-commerce website built using HTML, CSS, and JavaScript. The website is designed to provide a seamless and visually appealing shopping experience.

About the Project

This project aims to provide users with an efficient and enjoyable online shopping experience for sports equipment. With a clean and intuitive interface, users can easily browse and purchase products.

Key Features

  • Intuitive Navigation
  • Product Search and Filter
  • Shopping Cart
  • Secure Checkout
